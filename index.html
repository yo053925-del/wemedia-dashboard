CSV 批次匯入
貼上各平台匯出的 CSV，或直接複製數字貼入。支援一次匯入多個 IP 多個平台。
格式（逗號分隔，第一列為標題可省略）： IP名稱,平台,日期(YYYY-MM-DD),觀看數
T大的理財投資筆記,instagram,2025-04-20,125000
小貝財商,tiktok,2025-04-20,340000
林雨暘證券分析師,youtube,2025-04-20,89000
ZIP 內可包含多個 CSV 自動批次解析

YouTube
YouTube Studio → 數據分析 → 進階模式 → 右上角「匯出」→ 選時間範圍 → 下載 ZIP（含表格資料 + 圖表資料）

Meta Business Suite（business.facebook.com）→ 洞察報告 → 內容 → 選 Reels → 右上角「匯出」→ 下載 CSV，取「影片觀看次數」欄位

Google Sheets 串接設定

💡 取得方法：Google Sheet → 檔案 → 共用 → 發佈到網路 → 選「CSV」格式 → 複製連結
